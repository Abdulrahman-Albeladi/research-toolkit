examples/code_submissions

Place one or more .txt files here as inputs for the detection pipeline quickstart.

Formatting options:
- simplest: paste a single code snippet; it will be treated as main.txt
- multiple files: begin each section with a line that looks like a relative path to a code file (e.g., src/app/main.py or web/index.js) followed by its contents. Optional separator lines like ########## are allowed.

Do not include sensitive or proprietary material. This folder is versioned only as a scaffold; no real data is committed.
